# touch

`touch [OPTION]... FILE...` 创建新文件或修改时间戳

修改文件的访问时间与修改时间为当前时间；如果文件不存在将创建新的空文件

选项

```
-a                  只修改访问时间
-c                  不创建新文件
-m                  只修改修改时间
-d                  使用指定的时间而非当前时间
-r                  使用指定的文件的时间而非当前时间
-t                  使用 [[CC]YY]MMDDhhmm[.ss] 形式的日期而非当前时间
```
